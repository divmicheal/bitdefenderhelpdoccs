Managing Your Account Settings
===============================

After logging in, users can access multiple features within their dashboard.

Subscription Management
------------------------

• View active plans  
• Upgrade or renew subscription  
• Cancel auto-renewal  

Device Management
------------------

• Add new device  
• Remove inactive device  
• View protection status  

Security Settings
------------------

• Enable two-factor authentication  
• Update password  
• Review login activity
